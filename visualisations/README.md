# Interactive Visualisations

This folder contains interactive HTML demonstrations of the HIPSTer ontological framework.

## File Overview

| File | Content | Paper Reference |
|------|---------|-----------------|
| `ontology-network.html` | Core ontology structure | Figure 1 |
| `ontology-demo.html` | Russia-Ukraine case study | Section 4 Case Study |
| `sparql-demo.html` | Simplified SPARQL query demos | Section 4.7 (illustrative) |

## Important Notes

### Indicator Demonstration

The `ontology-demo.html` visualisation demonstrates **5 of the 7 indicator instances** shown in Figure 3:
- IDEO-14 (Coordinated Activity)
- 5COL-08 (Attribution Reversal)
- RAD-GEN-08 (Dehumanizing Metaphors)
- CONSP-12 (Hidden Enemy Narrative)
- DEHUM-01 (Demonizing Language)

The complete set of 143 indicator instances is available in the `/indicators/` folder.

### SPARQL Demonstrations

The interactive SPARQL demonstrations are **simplified versions** for accessibility.
For formal query specifications, see `/queries/`.

## Usage

Open any `.html` file in a modern web browser (Chrome, Firefox, Edge recommended).
